Frequently Asked Questions

If you consider yourself a frontend developer, here are some common questions and answers you should know.

Grid is made for two-dimensional layout while Flexbox is for one. Which means Flexbox can work on either row or columns at a time, but Grids can work on both. If you need to design any complex-layout gid is most preferable for that kind of work.

Bootstrap is component based framework and Tailwind is utility based. Component based means Bootstrap have lots of prebuild componet with utilities. You can use them by calling associative class. ailwind, on the other hand, CSS is a utility-first framework. Using Tailwind CSS is akin to writing regular CSS. Unlike Bootstrap, it has no prebuilt components.

The CSS box model is a container that contains multiple properties including borders, margin, padding, and the content itself. It is used to create the design and layout of web pages.

What are semantic tags in HTML? Image result for what is semantic tag A semantic element clearly describes its meaning to both the browser and the developer. Examples of semantic elements: [<-form> , <-table> , and <-article> ]- Clearly defines its content.
